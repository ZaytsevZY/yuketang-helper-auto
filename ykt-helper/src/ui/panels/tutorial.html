AI雨课堂助手使用教程

功能介绍

AI雨课堂助手是一个为雨课堂提供辅助功能的工具，可以帮助你更好地参与课堂互动。

项目仓库：GitHub

脚本安装：GreasyFork

工具栏按钮说明

  • 习题提醒：切换是否在新习题出现时显示通知提示（蓝色=开启）。
  • 课件浏览：查看课件与题目页面。
  • AI 解答：向 AI 询问当前题目并显示建议答案。
  • 自动作答：切换自动作答（蓝色=开启）。
  • 设置：配置 API 密钥与自动作答参数。
  • 使用教程：显示/隐藏当前教程页面。

自动作答

  • 在设置中开启自动作答并配置延迟/随机延迟。
  • 需要配置 DeepSeek API 密钥。
  • 答案来自 AI，结果仅供参考。

AI 解答

  1. 点击设置（）填入 API Key。
  2. 点击 AI 解答（）后会对“当前题目/最近遇到的题目”询问并解析。

注意事项

1) 仅供学习参考，请独立思考；2) 合理使用 API 额度；3) 答案不保证 100% 正确；4) 自动作答有一定风险，谨慎开启。
